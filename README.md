# Interview Practice App

An interview practice system built with Flutter.

## Overview
This app provides a categorized list of interview questions to help users practice for software engineering interviews. Users can browse questions by category, view answers, and use a dedicated practice mode to simulate an interview setting where answers are hidden by default.

## Features
- **Categorized Questions:** Browse questions grouped by topic (Data Structures, System Design, Behavioral, etc.).
- **Question List:** View all questions in a specific category.
- **Practice Mode:** A distraction-free mode to test your knowledge, revealing answers one by one.
- **Responsive Layout:** Adapts beautifully across mobile, tablet, and web interfaces.

## Setup Instructions
1. Ensure you have Flutter installed (SDK 3.7.2 or higher).
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the application.
